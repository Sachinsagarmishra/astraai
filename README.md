# 🕉️ Astrai - Your AI Spiritual Companion

Astrai is an AI-powered, comprehensive Vedic spiritual companion platform. Built using modern mobile and web technologies, Astrai guides users to connect with the divine through love and devotion, adhering to the core philosophy of **"मन से, डर से नहीं"** (through love and devotion, not fear).

---

## 📱 Platform Overview

Astrai is designed as a **mobile-first application** supported by a minimalist web presence:

*   **Mobile App (`/mobile`)**: Expo React Native application featuring:
    *   **Daily Panchang**: Calculated timings (Sunrise/Sunset, Abhijit Muhurta, Rahu Kaal, Tithi, Nakshatra).
    *   **Digital Mala (Jaap)**: Interactive SVG bead counter with haptic vibration feedback, target setups, and auto-chant routines.
    *   **AI Palmistry (Astro Lab)**: Multi-line analysis (Life, Heart, Head, Fate lines) using Gemini AI with base64 image capture.
    *   **Puja Vidhi Library**: Step-by-step guides, samagri checklist trackers, and mantra chant templates.
    *   **Profile & Streaks**: Progress tracking to motivate daily rituals.
*   **Backend API (`/server` & `/server.ts`)**: Express.js server connected to a SQLite database. Handles secure, server-side AI processing (Gemini 3.5 Flash) and saves palmistry readings securely.
*   **Landing Page Website (`/src`)**: Pure Vite + React + TailwindCSS marketing landing page, featuring required store policy guidelines (Privacy Policy, Terms of Service, Disclaimer).

## 🚀 Running Locally

Follow these steps to set up and run the backend, landing page, and mobile app.

### 1. Prerequisites
*   Node.js (v18+)
*   Gemini API Key from [Google AI Studio](https://aistudio.google.com/)

### 2. Installation
Install all dependencies in the root directory:
```bash
npm install
```

### 3. Database & Environment Configuration
1.  Create a `.env` file in the root directory and add your Gemini API Key:
    ```env
    GEMINI_API_KEY=your_gemini_key_here
    PORT=3001
    ```
2.  Initialize the SQLite database schema:
    ```bash
    npm run db:setup
    ```

### 4. Running the Backend & Landing Page
To start the backend API and the Vite website concurrently in development mode:
```bash
npm run dev
```
*   The website landing page will be served on `http://localhost:5173` (or the first available port).
*   The API server will listen on `http://localhost:3001` (redirecting web requests via Vite reverse proxy).

### 5. Running the Mobile App (Expo Go)
1.  Navigate to the `mobile/` directory:
    ```bash
    cd mobile
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the Metro bundler:
    ```bash
    npx expo start
    ```
4.  Open the **Expo Go** app on your physical iOS or Android device and scan the QR code printed in the terminal.
